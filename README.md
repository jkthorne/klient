# klient

DO NOT USE THIS.

I am experimenting on simplified clients

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     klient:
       github: jkthorne/klient
   ```

2. Run `shards install`

## Usage

```crystal
require "klient"
```
